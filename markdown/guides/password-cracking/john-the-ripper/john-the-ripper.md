
# John the Ripper

Named after the infamous Jack the Ripper, this tool for offline [password cracking](https://slavetomints.github.io/guides/vocabulary.html#password-cracking) is one of the best there is. From an extensive amount of supported [hashes](https://slavetomints.github.io/guides/vocabulary.html#hash), to a wide functionality thanks to [Jumbo John](https://slavetomints.github.io/guides/vocabulary.html##jumbo-john), John the Ripper is a flexible tool that will help you in your password cracking journey.
