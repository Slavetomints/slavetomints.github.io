
# Installation

> Disclaimer: this only covers Linux at the moment

If you use [Kali linux](https://slavetomints.github.io/guides/vocabulary.html#kali-linux), then you're all good! You don't need to install [John the Ripper](https://slavetomints.github.io/guides/vocabulary.html#john-the-ripper) since it comes preinstalled on your system. The same goes for [ParrotOS](https://slavetomints.github.io/guides/vocabulary.html#parrot-os), but you'll have to make an [alias](https://slavetomints.github.io/guides/vocabulary.html#alias) such as `alias john='/usr/sbin/john'` in `~/.bash_aliases` in order to call it on the command line. Otherwise ParrotOS allows you to call [johnny](https://slavetomints.github.io/guides/vocabulary.html#johnny), the GUI version of John the Ripper.

For most other modern distros, you can install `john` with your package manager, and you'll get John the Ripper that way.
